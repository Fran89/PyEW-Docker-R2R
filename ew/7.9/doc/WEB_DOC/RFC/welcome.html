EARTHWORM Request For Comments List
Update Feb 18, 1998

Introduction

In this section you will find documentation and descriptions of EARTHWORM modules which are currently being designed or developed. The description of the module is provided with the aim in mind of soliciting comments from others interested in this module or with ideas on what or how it should do things. Each of these Requests For Comments (RFC) is provided by an individule who is working on the module or is planning soon to be working on the module. Comments can be sent directly to that individual or to the general e-mail list for people interested in EARTHWORM, earthw-list@nmt.edu. Relevant coments sent to the whole list will be appended to the WEB RFC page.

To have an RFC started for a new proposed module, send a text message describing the module and its design issues to either the EWAB chairman, Ken Smith (ken@seismo.unr.edu), or to the EWAB WEB manager, Steve Malone (steve@geophys.washington.edu), or directly to the EARTHWORM mailing list at: earthw-list@nmt.edu
