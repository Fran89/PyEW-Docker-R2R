Eqalarm_ew Overview

(last revised 2 February, 1998)

This module serves to provide Earthworm with an internal minimalist notification capability for installations where there may not be any other notification software, and to serve as a pattern making it easier for such installations to create their own modules.

Eqalarm_ew attaches to the ring carrying 'eqproc' (hypoinverse) output. It listens for "TYPE_H71SUM" messages, and generates a "TYPE_PAGE" message if the event magnitude is greater than the threshold value given in the module's configuration file, "eqalarm_ew.d".

Note that "eqalarm_ew" uses the antiquated buffering scheme that requires a 'private' shared memory ring named "EQALARM_EW_RING". This must be declared in "startstop.d" and in "earthworm.d" as one of the rings in the system.

The format of the output message is given below. The first two lines are intended to show column number; the third line is the actual message text.

00000000001111111111222222222233333333334444444444555555555566666666667777777777

01234567890123456789012345678901234567890123456789012345678901234567890123456789
