Earthworm Modules:
SaraAdSend for Earthworm 7.x Overview

Created January 23 2008 by Larry Cochrane

SaraAdSend is another digitizing program similar to ADsend for Earthworm. This module runs under Windows 2000 or XP and produces demultiplexed data to one of the transport data rings. This program was designed to work with the SARA RS-232 serial output Analog to Digital boards.

This module was built using Microsoft Visual C++ 8.0. The program (SaraAdSend.exe) has been tested using Earthworm version 7.2.
