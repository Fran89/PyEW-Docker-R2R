Earthworm Program:
tbuf2mseed Overview

(last revised April 21, 2015)

The tbuf2mseed program reads TRACEBUF2 messages from one ring, converts them to MiniSEED messages, and deposits them into another ring. The earthworm_global.d file should have the TYPE_MSEED packet type defined. The tbuf2mseed binary is built via the ringdup code (with the "_TBUF2MSEED" symbol defined).
