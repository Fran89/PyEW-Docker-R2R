Status Overview

(last revised May 26, 2006)

This program shows the earthworm status, the same as if status was requested from startstop with a carriage return. This program is useful if there isn't access to the startstop console, or startstop is running without the console. The columns displayed vary slightly between platforms, here's an example from Windows:
