Earthworm Modules:
nq2gif Overview

(last revised 25 December, 2008)

nq2gif is a MiniSEED data file reading and plotting program for Earthworm. The utility was written for the No Cal Seis Net NetQuakes project.

nq2gif is designed as a multi-threaded program. The main thread loops polling the input directory, for new MiniSEED data files arriving from the NetWQuakes server. Once data arrives, the data is read and plotted in GIF format.
