Earthworm Program:
config_ws Overview

(last revised June 4, 2006)

The config_ws program listens to a WAVE_RING for TRACEBUF or TRACEBUF2 messages and writes the necessary tank files format for WaveServerV. This is useful if you have to sniff a ring and archive all of the trace's from that ring, but you don't know exactly what SCN/SCNL's are on it.

This module was contributed from the NEIC Hydra project by John Patton.

Warning: only tested under Windows.
