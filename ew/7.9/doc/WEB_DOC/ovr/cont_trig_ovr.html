Earthworm Modules:
Cont_trig Overview

(last revised July 12, 2011)

This module sends regularly trig2k messages in order for a listening trig2disk to produce continuous data archiving. It can replace the contrecord Perl and modules of GlowWorm package (used until v6.3) and of several other authors.

Based on carlsubtrig module.
