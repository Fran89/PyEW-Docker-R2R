Earthworm Modules:
Trig2arc Overview

(last revised July 12, 2011)

This module converts trig2k messages into hyp2000_arc messages. Only the stations that has been triggered are sent in the output. P arrival time is the trigger time for each station, and the phase weight is always 4.
