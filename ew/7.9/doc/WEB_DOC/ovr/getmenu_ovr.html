Getmenu Overview

(last revised 1 June 2006)

Program getmenu gets the menu from an Earthworm wave server.

getmenu <host>:<port>
Getmenu retrieves and prints the menu from the wave server running on host at a specific TCP port . The menu is a list of all the station-component-network-location (SCNL) combinations and their time period currently available from the wave server. This menu does NOT identify any gaps in the data at the wave server.

Host can be a hostname or an IP address.

Example command lines:

getmenu 192.168.5.200:16022
or
getmenu earthworm.usgs.gov:16022
or
getmenu localhost:16022
