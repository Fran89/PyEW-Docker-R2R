Earthworm Module:
Ewthresh Overview

(last revised 12 April, 2012)

The ewthresh module monitors a ring for TRACEBUF2X messages. If a message's maximum (converted) value exceeds the threshold specified for that message's SCNL, an alarm message is written to the output ring identifying the SCNL, threshold, time, and maximum (converted) value.

The primary option is what SCNLs to monitor and what threshold to use for it.

If the optional ThreshVotes command is included, then whenever the specified number of alarms have been triggered within the specified timeframe, a triglist message is generated, with a time of the specified amount before the first trigger of the group, and including the listed SCNLs

Note that the module ID of the output THRESHALARM message will be that of the TRACEBUF2X generating program, not the ewthresh module. This could be made an option at a later date.
