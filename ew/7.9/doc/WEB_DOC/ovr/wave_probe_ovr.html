Wave Probe Overview

(last revised 1 June, 2006)

Wave Probe is designed to send a trace request for one station to a waveserver, creating a debugging file for use in diagnosing waveserver problems.

This program is NOT location code compatible.

Usage: wave_probe <server> <port> <S> <C> <N> <yyyymmddhhmmss.ss>

Example:

wave_probe 127.0.0.1 16022 NP1 HHZ ZZ 20060601173455.00
