Earthworm Modules:
Coaxtoring Overview

(last revised 3 February, 1998)

This program copies Earthworm messages from Ethernet to local shared memory (known as a "transport ring"). Incoming messages may be of any installation id, message type, and module id, but they must have the same port number, specified in the configuration file. The program has three threads. The first thread (MsgIn) collects packets and assembles them into messages. The second thread writes messages to the transport shared memory ring buffer. The third thread generates heartbeat messages. Messages are transferred without translation of any kind.
