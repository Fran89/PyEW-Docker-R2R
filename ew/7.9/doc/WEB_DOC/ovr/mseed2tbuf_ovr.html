Earthworm Program:
mseed2tbuf Overview

(last revised April 21, 2015)

The mseed2tbuf program reads MiniSEED messages from one ring, converts them to TRACEBUF2 messages, and deposits them into another ring. The converted data will be split into two or more TRACEBUF2 packets as needed to make sure that they don't exceed the 4096-byte TRACEBUF2 size limit. The mseed2tbuf binary is built via the ringdup code (with the "_MSEED2TBUF" symbol defined).
