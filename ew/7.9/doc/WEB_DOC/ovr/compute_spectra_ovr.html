Earthworm Module:
Compute_spectra Overview

(last revised 5 May, 2011)

Deprecated as on 5/5/2011; use activate_module instead, using the same arguments as here but adding the module ID of your ewspectra module before the ring name.

The compute_spectra module write a message to the specified ring to signal the ewspectra module to process the specified timespan. All options are specified via the command line; its usage is as follows: 

compute_sepctra

where <start> is of the form YYYYMMDDHHMMSS, and <duration> is the number of seconds.
