Earthworm Modules:
Coda AAV Overview

(last revised 25 October, 2012)

This program produces 2s coda avg absolute values (aav). The program was decoupled from pick_ew so that picks and coda can potentially be produced from differently-processed waveform streams of a given SCNL.

Coda_aav only processes SCNLs listed in the pick_ew station list with pick_flag=1, and it uses only one parameter: 

              Old name   New name                                          
         --------   --------                                          
            c1       RawDataFilt

For more discussion of codas, see: pick_ew.scale-factors.txt
