Earthworm Program:
  wftimeshifter Overview

(last revised 3 December, 2011)
This module reads waveform data (compressed or uncompressed) from one transport ring and writes it to another ring, adding an offset to starttime and endtime. Intended to accomodate low-jitter, high-latency analog telemetry.
