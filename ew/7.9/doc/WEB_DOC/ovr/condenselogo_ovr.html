Earthworm Program:
condenselogo Overview

(last revised Dec 12, 2006)

The condenselogo program takes messages from one ring and rewrites them to a second ring with the logo of the module. It is useful for rewriting logo's of data source modules where the module ID's might conflict.

This module was contributed from the Menlo Park USGS group
