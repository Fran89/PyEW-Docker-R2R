Copystatus Overview

(last revised 27 July, 1998)

The status manager ("statmgr") program reads all error and heartbeat messages from a single transport ring (shared-memory region). However, client programs to the status manager may write their error and heartbeat messages to any transport ring. In order to make all error and heartbeat messages accessible to the status manager, the copystatus program is used to copy messages from one transport ring to another.
