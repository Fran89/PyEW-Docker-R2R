pick_recorder Overview

(last revised May 25, 2006)

The pick_recorder utility captures TYPE_PICK2K and TYPE_PICK_SCNL messages to a file per day. The user can specify the pick directory where the day files will be stored and they are named pick.YYYYMMDD with the UTC date of the picks contained therein.
