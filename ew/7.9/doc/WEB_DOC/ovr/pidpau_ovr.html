Earthworm Modules:
Pidpau Overview

(last revised May 25, 2006)

The pidpau command line utility requests that a Earthworm terminates the particular module associated with the PID (Process ID) you give pidpau as a parameter. The module should stop itself if it's behaving properly.

The program attaches to earthworm's shared memory region(s) and sets the flag in the header to the given value (pid). This is intended to signal that processid to terminate gracefully.
