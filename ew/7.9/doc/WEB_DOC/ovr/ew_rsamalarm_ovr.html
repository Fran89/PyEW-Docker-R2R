Earthworm Module:
Ew_rsamalarm Overview

(last revised 11 April, 2012)

The ew_rsamalarm module is an RSAM monitor: it reads TRACEBUF and TRACEBUF2 messages as would be produced by ew2rsam; recognizes when identified stations have an event, logging and producing messages with statistics on each event; groups stations into subnets; when enough stations in a subnet participate in an event concurrently, an alarm message is produced.
