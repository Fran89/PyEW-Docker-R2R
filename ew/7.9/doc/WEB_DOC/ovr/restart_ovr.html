Earthworm Program:
Restart Overview

(last revised June 28, 2011)

The restart program sends an Earthworm TYPE_RESTART message with the specified pid(s) to startstop, for manual restarting of individual modules. The pid can be obtained from the status message printed by status.

The module writes the TYPE_RESTART for each pid provided into the first ring created by startstop. For this reason, the startstop file must be named as per OS naming conventions, or passed in as the second argument after a -c flag is given. The official usage echoed back is provided below: 

$ restart
Usage: restart [-c config_file]  [ ...]
