Reconfigure Configuration File

(last revised May 25, 2006)

Reconfigure looks for changes in startstop's configuration file, startstop*.d.

Reconfigure is invoked from within startstop with the command:

reconfigure

(this can be abbreviated recon)

Reconfigure is invoked from the command line (provided you have the earthworm environment variables set from the appropriate script in the environment directory) by typing:

reconfigure
