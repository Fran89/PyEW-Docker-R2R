Network Characteristics

Information listed below includes both the Utah and Yellowstone networks

  • ~100 analog, short-period telemetry stations (~160 channels)
  • 5 broadband stations with analog telemetry
  • 2 broadband stations using digital telemetry, 2 additional to be installed shortly
  • 2 strong motion stations (one hardwire connection and one with cell phone dial-up access)
  • 7 USNSN broadband stations in monitoring region
  • Analog network data digitized by Concurrent 7200C computer using HAWK event detection software
  • Continous telemetery from 24-bit Reftek digitizers

Non-Standard Earthworm Modules in current use

  • reftoew: receives packets from Reftek dataserver (dataserver software from UNR), puts data into EW trace_ring, and outputs continuous SAC files. Code written by D. Chavez under contract to the University of Utah.
  • nsn_rcv: receives data feed from USNSN (currently 7 stations located in montoring area). Code written by D. Chavez under contract to the University of Utah.
  • nsn_sta: writes SAC-format files for USNSN data locally. Code written by D. Chavez under contract to the University of Utah..
  • vdl: sends data feed (currently 8 analog-telemetered, short-period and 2 digitally-telemetered BB stations) to USNSN. Code written by D. Ketchum, USGS.
