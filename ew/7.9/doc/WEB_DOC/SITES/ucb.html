Earthworm Modules under development

  • Use binder to associate events from BDSN Murdock-Hutt detector messages, which are imported as EW picks (pick2 format). Developed MHH->Pick import module.
  • Experimental 3-component broadband phase detector module.
  • Exchange waveforms between BDSN and NCSN using binary import/export. Developed a comserv/ew module for MiniSEED->EW waveform import.
  • Developed EW MiniSEED waveform format and import procedure for MiniSEED waveform data.
  • Utilize EW rings for monitoring hearbeats of various aspects of REDI processing.
