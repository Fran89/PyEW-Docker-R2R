EARTHWORM DISTRIBUTION

 

How to get the distribution and documentation (version 6.3 or greater)

  • The software distribution is available via FTP: ftp://anonymous%40isti2.com@ftp.isti2.com/pub/ew/
    If you're asked for a password, leave it blank and click OK. For Safari, try the direct link below, or change the %40 to an @ in the username when the window pops up.
    (The username to get into ftp.isti2.com is "anonymous@isti2.com". That user has no password. Yes the "@" is part of the user name... The above URL with %40 substituted for the @ should work if you're using a web browser that can do ftp://.)
    OR try the direct link to our ftp server: ftp://ftp.isti2.com/pub/ew
  • The software distribution is available via HTTP: http://folkworm.ceri.memphis.edu/ew-dist
    OR http://www.isti2.com/ew/distribution
